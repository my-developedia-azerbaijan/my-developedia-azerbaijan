# Başlıq: Malware nədir?
**Malware (zərərli proqram)** istifadəçinin razılığı olmadan sistemdə arzuolunmaz və zərərli əməliyyatlar aparan proqram təminatıdır. Adi proqramlardan fərqli olaraq, o çox vaxt gizli işləyir və öz mövcudluğunu gizlətməyə çalışır.
Malware sistemlərə ziyan vurmaq, məlumat oğurlamaq, xidmətləri pozmaq və ya cihazlara və şəbəkələrə icazəsiz giriş əldə etmək məqsədi ilə yaradılmış hər hansı proqram və ya koddur. Malware mövzusunu anlamaq kibertəhlükəsizlik mütəxəssisləri, proqramçılar və adi istifadəçilər üçün rəqəmsal mühitlərin qorunması baxımından çox vacibdir.

### Malware-in əsas məqsədləri:
- Şəxsi və ya maliyyə məlumatlarını oğurlamaq
- İstifadəçini gizli şəkildə izləmək
- Faylları zədələmək və ya şifrələmək
- Cihazı ələ keçirmək
- Şirkət və xidmətlərin işini pozmaq

### Malware növləri:

- **Virus:** Virus qanuni fayllara qoşulur və həmin fayllar paylaşıldıqda və ya işə salındıqda yayılır. Adətən aktivləşmək üçün istifadəçi əməliyyatı tələb edir.
- **Worm (Soxulcan):** Soxulcanlar istifadəçi müdaxiləsi olmadan şəbəkə üzərindən avtomatik yayıla bilir. Onlar zəifliklərdən istifadə edərək öz surətlərini başqa cihazlara köçürürlər.
- **Trojan (Troya atı):** Trojan faydalı proqram kimi görünür, lakin quraşdırıldıqdan sonra gizli zərərli əməliyyatlar icra edir.
- **Ransomware:** Ransomware qurbanın fayllarını bloklayır və ya şifrələyir, bərpa üçün ödəniş tələb edir.
- **Spyware:** Spyware klaviatura yazılarını, brauzer tarixçəsini və ekran fəaliyyətini gizli şəkildə izləyir.
- **Adware:** Adware arzuolunmaz reklamlar göstərir və istifadəçi davranışını izləyə bilər.
- **Rootkit:** Rootkit əməliyyat sisteminin dərinliklərində gizlənir və aşkarlanmanın qarşısını alaraq hücumçuya uzunmüddətli nəzarət verir.
- **Botnet:** Yoluxmuş cihazlar bir şəbəkəyə – botnetə qoşularaq uzaqdan idarə edilir və kütləvi hücumlar üçün istifadə olunur.


### Malware necə yayılır?:

- E-poçt qoşmaları və fişinq keçidləri
- Saxta proqram yükləmələri
- Yoluxmuş USB daşıyıcılar
- Proqram zəifliklərinin istismarı
- Zərərli veb saytlar
- Sosial mühəndislik mesajları

### Malware-in həyat dövrü:

1. **Çatdırılma** – zərərli kod hədəfə çatdırılır.
2. **İcra** – proqram cihazda işə düşür.
3. **Davamlılıq** – yenidən başladıqdan sonra qalmağa çalışır.
4. **Fəaliyyət** – oğurluq, şifrələmə və ya casusluq.
5. **Yayılma** – başqa sistemlərə keçməyə cəhd.


### Yoluxma əlamətləri:

- Kompüterin yavaşlaması
- Naməlum proqramların görünməsi
- Çoxlu pop-up reklamlar
- Faylların dəyişməsi və ya şifrələnməsi
- Səbəbsiz internet trafiki

### Müdafiə tədbirləri:

- **İstifadəçi vərdişləri:**
    - Şübhəli link və faylları açmamaq
    - Yalnız etibarlı mənbələrdən yükləmə
    - USB cihazlara diqqətli yanaşmaq

- **Texniki qoruma:**
    - Sistemləri daim yeniləmək
    - Etibarlı antivirus istifadə etmək
    - Firewall aktiv saxlamaq
    - Müntəzəm ehtiyat nüsxə yaratmaq
    - Güclü autentifikasiya tətbiq etmək

- **Təşkilatlar üçün:**
    - Təhlükəsizlik təlimləri
    - Şəbəkə monitorinqi
    - Giriş nəzarəti
    - İnsidentə cavab planı



## Nəticə:
Malware rəqəmsal dünyanın ən böyük təhlükələrindən biridir. Onun iş prinsiplərini anlamaq məlumat və sistemlərin qorunması üçün vacibdir. Maarifləndirmə, yeniləmələr və diqqətli davranış ən güclü müdafiə üsullarıdır.
